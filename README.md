# NewsAlphaAI

AI-powered stock prediction system that analyzes financial news and market data to forecast short-term stock movement.

The system uses LLM reasoning, RAG pipelines, vector databases, and reinforcement learning to identify how news events impact stock prices and predict whether a stock will move **UP or DOWN within the next 2–3 days**.

By combining real-time news sentiment analysis with market indicators, the model continuously improves its predictions through feedback from actual market outcomes.

## Key Features

• Real-time financial news ingestion
• LLM-based sentiment and impact analysis
• Vector database powered RAG retrieval
• LangGraph AI pipeline orchestration
• Stock data integration and feature extraction
• ML prediction models for price direction
• Reinforcement learning feedback loop
• Explainable AI predictions

## Tech Stack

Python • LangChain • LangGraph • LangSmith
Vector Databases (Chroma / FAISS)
LLMs (OpenAI / LLaMA / Mistral)
Scikit-learn • XGBoost • PyTorch
News APIs • Stock APIs

## Goal

Build an AI system that understands how news affects financial markets and generates actionable trading insights.
